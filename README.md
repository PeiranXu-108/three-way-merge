# Three-Way Diff Workspace

This repo contains an open-source three-way diff editor package.

## Structure

- `packages/three-way-diff` - reusable React component library
- `apps/demo` - Vite demo site

## Development

- `npm install`
- `npm run dev`
- `npm run storybook`

## Build

- `npm run build`
- `npm run build-storybook`
